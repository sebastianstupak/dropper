# Dropper Examples

This directory contains example projects generated by Dropper for testing and demonstration purposes.

## test-dev-mod

Generated by `FullCLIBuildTest` E2E test to verify that:
- Project generation works correctly
- File structure is properly created
- ConfigReader can parse the project
- GradleRunner can construct correct commands
- Dev command infrastructure is set up

### Setup

To test this project manually:

```bash
cd test-dev-mod

# Create Gradle wrapper (required for first-time setup)
gradle wrapper

# List available tasks
./gradlew tasks

# Build the project
./gradlew build

# Run with Fabric on 1.20.1
./gradlew :1_20_1-fabric:runClient

# Run with NeoForge on 1.21.1
./gradlew :1_21_1-neoforge:runClient
```

### Testing Dev Command

Once gradlew is set up, you can test the dev command:

```bash
# Auto-detect version/loader
dropper dev run

# Specific version/loader
dropper dev run --version 1.20.1 --loader fabric

# Debug mode
dropper dev run --debug --port 5005
```

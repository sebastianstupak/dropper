# Minecraft Block Textures

This directory contains Minecraft block textures for the Dropper landing page.

## Required Images

Download these PNG files (16x16 or 64x64) and place them in this directory:

1. **dropper.png** - Dropper block (main logo)
2. **crafting_table.png** - Crafting table
3. **furnace.png** - Furnace
4. **chest.png** - Chest
5. **grass_block_side.png** - Grass block
6. **stone.png** - Stone block
7. **dirt.png** - Dirt block
8. **diamond_block.png** - Diamond block

## Where to Download

### Official Minecraft Assets
- **MC Asset Cloud**: https://mcasset.cloud/
  - Select Java Edition > Latest Version
  - Navigate to textures/block/
  - Download individual block textures

### Community PNG Sites (Transparent backgrounds)
- **NicePNG**: https://www.nicepng.com/s/minecraft-block/
- **PNGkey**: https://www.pngkey.com/pngs/minecraft-block/
- **KindPNG**: https://www.kindpng.com/free/minecraft-block/
- **StickPNG**: https://www.stickpng.com/img/games/minecraft/block-minecraft

### Minecraft Wiki
- **Block Textures**: https://minecraft.wiki/w/Category:Block_textures
- Individual PNG files for each block

## Usage

Once downloaded, these images will be used in:
- Hero section logo (dropper.png)
- Feature icons (various blocks)
- Background textures
- Decorative elements
